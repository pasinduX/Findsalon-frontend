# Fidsalon Next

## Project info

A Vite + React + TypeScript starter project with Tailwind CSS.

## How do I edit this code?

Edit the source files locally in your preferred IDE.

### Local development

Install dependencies and start the dev server:

```sh
npm install
npm run dev
```

### Recommended workflow

- Open the project folder in your editor
- Modify files in `src/`
- Refresh the browser or let Vite reload automatically
- Commit and push changes using Git

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the app and deploy the generated `dist/` folder using your preferred hosting provider.

```sh
npm run build
```

## Notes

This repository no longer includes external helper configuration or branding.
